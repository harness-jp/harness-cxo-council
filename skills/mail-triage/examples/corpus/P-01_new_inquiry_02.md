# P-01: 新規問合せ・初動対応 — 例2（名刺交換後フォロー）

## 受信メール

```
件名: 先日は名刺交換ありがとうございました
From: sato@example.org
To: contact@example.org
日時: 2026-05-21 16:45
```

```
ご担当者様

先日の業界カンファレンスにて名刺交換させていただきました、
○○ホールディングスの佐藤と申します。

その節は貴重なお話をお聞かせいただき、ありがとうございました。

ご紹介いただいた事例について、改めて詳しく伺いたく、
オンラインでも構いませんので、面談のお時間を頂戴できればと思います。

何卒よろしくお願い申し上げます。

○○ホールディングス
佐藤花子
sato@example.org
```

## 期待される処理

- **PB**: P-01（新規問合せ・初動対応）
- **信頼度**: 0.92
- **判定根拠**:
  - 件名「名刺交換」+ 本文「先日のカンファレンスで」= 過去接点ありの新規アクション
  - 送信者は名刺交換済だが、メールでのやり取りは初回
- **skip判定**:
  - L1 越境誤爆: 通過
  - L2 沈黙: 通過（新規スレッド）
  - L3 上から目線: draft 後に評価
- **no_autosend**: false（自動送信可）
- **PRECEDENCE**: 単独該当

## 期待される返信ドラフト

```
○○ホールディングス 佐藤様

ご連絡ありがとうございます。
カンファレンスでお話しできて嬉しく思っておりました。

事例の件、改めて詳しくお伝えできればと思います。

オンラインで承知しました。
5/28(木) 15:00-、5/29(金) 10:00- などはいかがでしょうか。
60分程度を想定しています。

よろしくお願いいたします。

{SIGNATURE}
```

## カスタマイズメモ

- 名刺交換ありの場合、初回ほどフォーマルにしすぎない（過剰丁寧は逆に距離感）
- 「貴社」より「○○ホールディングス」と固有名詞で呼ぶ方が温度感が出る
- 「お聞かせ」「お伝え」は中立的トーン。装飾語回避
