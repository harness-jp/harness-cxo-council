# P-05: 日程調整 — 例1（候補日提示要請）

## 受信メール

```
件名: 来週の打合せ候補日のご相談
From: kobayashi@example.org
To: contact@example.org
日時: 2026-05-27 09:00
```

```
ご担当者様

お世話になっております。
○○ホールディングス小林です。

先日ご相談いたしました件、来週中にお打合せの機会を頂戴できればと思います。
以下の候補日からご都合のよい日時をお知らせください:

- 6/3(火) 10:00-11:00
- 6/4(水) 14:00-15:00
- 6/5(木) 13:00-14:00

オンライン（Teams）を想定しています。
URLは確定後にお送りします。

よろしくお願いいたします。

○○ホールディングス 小林六郎
```

## 期待される処理

- **PB**: P-05（日程調整）
- **信頼度**: 0.96
- **判定根拠**:
  - 件名「候補日」+ 本文に複数候補日（3つ）+ 「ご都合のよい日時」
  - 物理的な現場立会ではなく打合せ調整 = P-04 ではなく P-05
- **skip判定**:
  - L1 越境誤爆: 通過（自分宛アクション要求）
  - L2 沈黙: 通過（新規依頼）
  - L3 上から目線: draft 後に評価
- **no_autosend**: false（候補日選択の返信）
- **PRECEDENCE**: P-04 と紛らわしいが「立会」「現場」キーワードなし → P-05

## 期待される返信ドラフト

```
○○ホールディングス 小林様

ご連絡ありがとうございます。

候補日の中では、6/4(水) 14:00-15:00 が都合よろしいです。

URLお待ちしております。

{SIGNATURE}
```

## カスタマイズメモ

- 1つ確定なら**1つだけ提示**（自分の予定が複数空いてても「全部空いてます」とは書かない）
- カレンダー登録の意思表示として「URLお待ちしてます」を入れる
- 「お忙しいところ恐縮ですが」のような前置きは入れない（依頼を受けるだけなので不要）
