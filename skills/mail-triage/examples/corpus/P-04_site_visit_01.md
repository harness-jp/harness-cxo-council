# P-04: 現場確認・引渡し — 例1（立会日程確定）

## 受信メール

```
件名: 6/3立会の件
From: watanabe@example.org
To: contact@example.org
日時: 2026-05-25 10:20
```

```
ご担当者様

お世話になっております。
○○ホールディングス渡辺です。

来週の現場立会の件、以下で確定でよろしいでしょうか:

日時: 6/3(火) 10:00-11:00
場所: 東京都○○区 ××ビル 3階
立会者: 弊社渡辺、貴社1名
持ち物: 立会記録様式、印鑑（朱肉付き）

当日は1階受付にて入館手続きをお願いいたします。
変更事項あれば、前日までにご連絡ください。

よろしくお願いいたします。

○○ホールディングス 渡辺四郎
```

## 期待される処理

- **PB**: P-04（現場確認・引渡し）
- **信頼度**: 0.96
- **判定根拠**:
  - 件名「立会」+ 本文に日時・場所・立会者・持ち物の4要素揃う
  - 「立会」は強い P-04 シグナル
- **skip判定**:
  - L1 越境誤爆: 通過（自分宛確定アクション）
  - L2 沈黙: 通過（新規確認）
  - L3 上から目線: draft 後に評価
- **no_autosend**: false（受領確認返信）
- **PRECEDENCE**: 単独該当（日時情報あるが P-05 ではなく P-04 が主）

## 期待される返信ドラフト

```
○○ホールディングス 渡辺様

ご連絡ありがとうございます。

6/3(火) 10:00-11:00、××ビル3階での立会、承知しました。
立会記録様式と印鑑を持参いたします。

当日よろしくお願いいたします。

{SIGNATURE}
```

## カスタマイズメモ

- 立会系は**事実情報の確認**が主目的。装飾語ほぼ不要
- 「変更時は連絡」のような将来事項は今回返信に含めない（不要な往復を増やす）
- 持ち物リストは原文のまま復唱せず「印鑑を持参」など短くまとめる
