# P-03: 担当者変更・引継ぎ — 例1（異動引継ぎ）

## 受信メール

```
件名: 担当者変更のご連絡
From: suzuki@example.com
To: contact@example.org
Cc: yamada@example.com
日時: 2026-05-22 11:15
```

```
ご担当者様

平素より大変お世話になっております。
ABC商事の鈴木です。

このたび社内異動に伴い、5/31をもって本件の担当を外れることとなりました。
6/1からは、新担当の山田（CCに記載）が引き継いで対応させていただきます。

直近の状況は以下の通りです:
- 4月見積の件: 山田より改めてご連絡予定
- 5/15打合せ議事録: 共有済（山田にも転送済）

お手数おかけしますが、引き続きよろしくお願いいたします。

ABC商事 鈴木一郎
```

## 期待される処理

- **PB**: P-03（担当者変更・引継ぎ）
- **信頼度**: 0.97
- **判定根拠**:
  - 件名「担当者変更」+ 本文「異動」「担当を外れる」「新担当」
  - CC に新担当者 (yamada@) が含まれている
- **skip判定**:
  - L1 越境誤爆: 通過（自分宛 To）
  - L2 沈黙: 通過（新規話題）
  - L3 上から目線: draft 後に評価
- **no_autosend**: false（受領 + 引継ぎ確認の返信）
- **PRECEDENCE**: 単独該当

## 期待される返信ドラフト

```
ABC商事 鈴木様
（CC: 山田様）

ご連絡ありがとうございます。
鈴木様のご異動、お疲れさまでした。

引継ぎ内容、承知いたしました。
山田様にも本メールにて挨拶代わりにご連絡となりますが、
今後ともよろしくお願いいたします。

直近の見積の件、山田様からのご連絡をお待ちしております。

{SIGNATURE}
```

## カスタマイズメモ

- 異動先・新担当の双方に丁寧に挨拶する形にする
- CC に新担当が入っていれば「（CC: 山田様）」と明示
- 「お疲れさまでした」は同等関係なら使える。上下関係が強いなら「ありがとうございました」に
