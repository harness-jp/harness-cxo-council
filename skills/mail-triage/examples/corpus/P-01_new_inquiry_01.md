# P-01: 新規問合せ・初動対応 — 例1（初コンタクト）

## 受信メール

```
件名: 貴社サービスについてのお問合せ
From: tanaka@example.com
To: contact@example.org
日時: 2026-05-20 09:32
```

```
ご担当者様

初めまして、ABC商事の田中と申します。
貴社ホームページを拝見し、サービス内容について詳しく伺いたく、
ご連絡いたしました。

弊社では現在、業務効率化のためのソリューションを検討しており、
貴社サービスがフィットするのではないかと考えております。

ご都合のよろしいタイミングで、30分ほどお時間いただけますでしょうか。

よろしくお願いいたします。

ABC商事 田中太郎
03-XXXX-XXXX
tanaka@example.com
```

## 期待される処理

- **PB**: P-01（新規問合せ・初動対応）
- **信頼度**: 0.95
- **判定根拠**:
  - 件名「お問合せ」+ 本文「初めまして」+ 自社サービスへの問合せ
  - 送信者ドメインが既存取引先リストに無い（新規）
- **skip判定**:
  - L1 越境誤爆: 通過（ball_with なし、自分宛）
  - L2 沈黙: 通過（新規スレッド）
  - L3 上から目線: draft 後に評価
- **no_autosend**: false（自動送信可）
- **PRECEDENCE**: 単独該当（他PBには該当しない）

## 期待される返信ドラフト

```
ABC商事 田中様

お問合せありがとうございます。

貴社の業務効率化のご検討、承知しました。
ご相談内容を伺いまして、フィットするかを一緒に確認させていただければと思います。

来週前半でしたら、5/26(火) 14:00-、5/27(水) 10:00- あたりでお時間取れます。
ご都合いかがでしょうか。

よろしくお願いいたします。

{SIGNATURE}
```

## カスタマイズメモ

- `{SIGNATURE}` は自社の署名に置換
- 候補日3つ提示が目安。少ないと再調整、多いと選びづらい
- 「ご連絡ください」を末尾に書かない（相手が既に連絡してきた）
